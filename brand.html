Typeface

Serif Typeface: Sanchez

  • The serif typeface is the primary typeface to be used on all assets including web, print, signage, etc.
  • Link: https://fonts.google.com/specimen/Sanchez
  • CSS Import: @import url('https://fonts.googleapis.com/css2?family=Sanchez:wght@800&display=swap');
  • CSS: font-family: 'Sanchez', serif;

Sans-Serif Typeface: League Spartan

  • The sans-serif typeface is an alternate typeface to be used where required, primarily as body text on web.
  • Link: https://fonts.google.com/specimen/League+Spartan
  • CSS Import: @import url('https://fonts.googleapis.com/css2?family=Spartan:wght@800&display=swap');
  • CSS: font-family: 'League Spartan', sans-serif;
